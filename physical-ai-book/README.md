# Physical AI & Humanoid Robotics Educational Book

This repository contains a comprehensive educational book on Physical AI and Humanoid Robotics, designed for advanced undergraduate and graduate students with a background in Python and machine learning fundamentals.

## Overview

This curriculum provides a complete 13-week program that progresses from foundational concepts to advanced implementations in robotics, covering:

- ROS 2 fundamentals and environment setup
- Physics-accurate robot simulations with Gazebo
- Isaac ROS perception and navigation systems
- VLA (Vision-Language-Action) models integration
- Capstone project integrating all concepts

## Features

- Hands-on examples with code snippets
- Simulation-first approach with real hardware deployment paths
- Troubleshooting sections for common setup issues
- Progressive difficulty curve from basic concepts to advanced implementations
- Three-tier hardware approach (Budget/Standard/Premium)

## Prerequisites

- Python programming experience
- Basic machine learning knowledge
- Ubuntu 22.04 LTS (primary target OS)
- ROS 2 Humble Hawksbill (LTS release)
